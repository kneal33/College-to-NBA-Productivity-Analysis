# College-to-NBA-Productivity-Analysis
Analyzed NBA player data (2014–2024) to evaluate which U.S. colleges produce the most productive NBA talent.

Built a full data pipeline and interactive Streamlit dashboard to rank colleges using advanced metrics such as True Shooting %, Defensive Impact, Effective FG %, Assist-to-Turnover ratio, and Rebounding.

## Project Overview
This project combines NBA player statistics with draft and college data to evaluate which U.S. colleges produce the most productive NBA players. Metrics include offensive efficiency, defensive contribution, and overall impact. Player stats are aggregated by college to rank programs based on total NBA productivity rather than just player count.

## Features
- Aggregates NBA player performance metrics by college
- Ranks colleges based on total NBA productivity
- Interactive visualizations with Streamlit
- Reproducible data pipeline for analysis

## Data
- NBA player stats (2014–2024)  
- Draft and college information
